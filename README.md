# Fake News Detector

This is a **Fake News Detection System** that I built while following a Machine Learning tutorial. The project demonstrates how machine learning and natural language processing (NLP) can be used to classify news articles as **Real** or **Fake** based on their textual content.

The application processes news articles by cleaning and transforming the text into numerical features using **TF-IDF Vectorization**. A machine learning model is then trained on a labeled dataset to predict whether a given news article is genuine or fake. This project helped me understand the complete machine learning workflow, from data preprocessing to model training, evaluation, and deployment.

## Features

- 📰 Detects whether a news article is **Real** or **Fake**
- 🧹 Text preprocessing and data cleaning
- 🔤 TF-IDF Vectorization for feature extraction
- 🤖 Machine Learning model for classification
- 📈 Model evaluation using accuracy and classification metrics
- 💻 Simple web interface for testing custom news articles
- ⚡ Fast prediction on user input

## Machine Learning Workflow

1. Load and combine the real and fake news datasets.
2. Clean and preprocess the text data.
3. Convert text into numerical features using TF-IDF Vectorizer.
4. Split the dataset into training and testing sets.
5. Train the machine learning model.
6. Evaluate the model using performance metrics.
7. Save the trained model for future predictions.
8. Connect the model to a Flask backend and web interface for real-time predictions.
